## 下载

### Docker

```
docker pull apachecn0/cython-doc-zh
docker run -tid -p <port>:80 apachecn0/cython-doc-zh
# 访问 http://localhost:{port} 查看文档
```

### NPM

```
npm install -g cython-doc-zh
cython-doc-zh <port>
# 访问 http://localhost:{port} 查看文档
```
