## 下载

### Docker

```
docker pull apachecn0/numba-doc-zh
docker run -tid -p <port>:80 apachecn0/numba-doc-zh
# 访问 http://localhost:{port} 查看文档
```

### PYPI

```
pip install numba-doc-zh
numba-doc-zh <port>
# 访问 http://localhost:{port} 查看文档
```

### NPM

```
npm install -g numba-doc-zh
numba-doc-zh <port>
# 访问 http://localhost:{port} 查看文档
```
