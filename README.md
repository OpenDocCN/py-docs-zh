## 下载

### Docker

```
docker pull apachecn0/py-docs-zh
docker run -tid -p <port>:80 apachecn0/py-docs-zh
# 访问 http://localhost:{port} 查看文档
```

### NPM

```
npm install -g py-docs-zh
py-docs-zh <port>
# 访问 http://localhost:{port} 查看文档
```
